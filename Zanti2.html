Description


zANTI2 is a mobile penetration testing toolkit that lets security managers assess the risk level of a network with the push of a button. This easy to use mobile toolkit enables IT Security Administrators to simulate an advanced attacker to identify the malicious techniques they use in the wild to compromise the corporate network.


It Uncover authentication, backdoor, and brute-force attacks, DNS and protocol-specific attacks and rogue access points using a comprehensive range of full customizable network reconnaissance scans.


It Enable Security Officers to easily evaluate an organizationâ€™s network and automatically diagnose vulnerabilities within mobile devices or web sites using a host of penetration tests including, man-in-the-Middle (MITM), password cracking and metasploit.


You can Highlight security gaps in your existing network and mobile defenses and report the results with advanced cloud-based reporting through zConsole. it mirrors the methods a cyber-attacker can use to identify security holes within your network. Dash-board reporting enables businesses to see the risks and take appropriate corrective actions to fix critical security issues.

cyborg@cyborg:~$ sudo zanti

after successful connecting your device run here 'adb install zANTI2.apk'


Install APK to Device :


root@cyborg:/pentest/forensics/zanti# adb install zANTI2.apk


5570 KB/s (14052429 bytes in 2.463s)


Success


Start Zanti on Device :

zant2 start Zanti2


Zanti will ask for Root Privilege :

zanti2 root Zanti2


Enter your ID to start Zanti :

zanti id Zanti2


Zanti Will Map Network :

zanti2 start Zanti2


Select Network to start with :

zanti2 select Zanti2


Meanwhile to can use the sidewise menu for Different tasks :

zanti2 menu Zanti2
